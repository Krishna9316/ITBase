Defination  => Lists are used to store multiple items in an single Variable
            => Liists are Changable
            => It's allows Duplicate Values
            => Lists are Ordered 
            => This are Indexing Based

Syntax :- thisList = ["Value1","Value2",2,2.5,True]

Methods of List
1. append()     => used to add elements to end of the list
2. copy()       => Return a Shallow copy of the list
3. color()      => Remove the all elements from list
4. count()      => return the number of times a specified element appears in the list
5. extend()     => adds elements from another list to the end of the current list
6. index()      => Return the index number of the first occurance of a specified element
7. pop()        => Remove the return the element at the specified position
8. Remove()     => remove the first occurance of a specified element
9. reverse()    => Reverse the order of the elements in the list
10. sort()      => sorts the list in ascending order.
