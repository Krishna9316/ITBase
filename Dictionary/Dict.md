Introduction :- Dictionary is a inbuild Data structure that stores a data in Key Value Pair. Where each eachh key is unique and is used to get its Value.
    => Dict is a Ordered
    => Dict is Mutable (Changable)
    => Dict is used unique keys to store data.
    => Dict is Constructed by using *dict()* or using curly Braces *{}*.
    

    eg. Using dict()
        => exampleList = dict(Name : "Krishna")
            print(exampleList)   //output : {'name' :'Krishna'}

        using Curly Braces {}
        => exampleList = {"Name" : "Krishna"}
            print(exampleList)   //output : {'name' :'Krishna'}
